# Product Definition - Contextware

## Vision
Contextware is an advanced agent skill designed to serve as the persistent cognitive layer for AI software engineering assistants. It captures, organizes, and retrieves critical context to transform ephemeral interactions into a continuous, coherent development lifecycle.

## Target Users
- **AI Agents/Assistants:** Engineering assistants that require a persistent memory layer to maintain state across sessions.

## Core Goals
- **Performance Benchmarking:** Test the agent skill against a suite of example codebases to measure its performance impact compared to re-running the same prompts without Contextware.
- **Context Preservation:** Bridge the gap between stateless AI operations and complex, long-running software projects.

## Key Features
- **Deep File Intelligence:** Indexing code and generating semantic summaries to provide high-level context beyond simple syntax.
- **Conversation Memory:** Recording and retrieving past chat interactions to maintain the "why" behind project decisions.

## Constraints
- **Lightweight Python Integration:** Must run efficiently within the Python 3.13 ecosystem and integrate seamlessly with existing tools.
- **Scalability to Large Repositories:** Engineered to handle the complexity and size of large enterprise-scale codebases without performance degradation.
